结论:
1、dom动画建议anime.js，语法简单上手快
2、konva语法简单，上手快，图形绘制、事件绑定容易，有图层及组概念，适用效果简单的小游戏，不支持IE11，支持edge
3、pixi.js有文件预加载接口，默认使用WebGL，当WebGL不支持时使用canvas，也可使用设置类型，可实现碰撞检测、骨骼动画、3D动画，添加库控制音效，更适合复杂类网页游戏
4、动画性能优劣：webgl > canvas > css + dom > js + dom

Event

new Event(type,eventInit);
type:创建事件的名称，DOMString类型
eventInit：可选配置(bubbles/cancelable/composed)

UIEvent

UIEvent 接口表示简单的用户界面事件，是从 Event 派生出来的，可以传detail字段（类型只能为数字,非数字时接收到的值为0）

CustomEvent

new CustomEvent(type,eventInit)
type:创建事件的名称，DOMString类型
eventInit：可选配置(bubbles/cancelable/detail)

Event、UIEvent、CustomEvent的区别

UIEvent、CustomEvent可以传递自定义参数detail，UIEvent detail类型只能为数字，CustomEvent可以是数字、字符串、数组、对象等

Electron预加载JS

发表于 2021-11-06 更新于 2022-04-12 分类于 Electron

在 index.js 主进程中，无法操作渲染进程的DOM，因为它无法访问到渲染进程 document 上下文，它们存在于完全不同的进程中。
为解决这个问题可使用预加载脚本，预加载脚本可同时运行于主进程和渲染进程。预加载脚本在渲染进程加载之前加载，并可以访问渲染进程 (例如 window 和 document) 和 Node.js 环境及Node.js API。在预加载脚本中给 window 挂载变量需开启contextIsolation。

Android手机adb工具使用

发表于 2021-07-28 分类于 移动端

测试手机：android 9 ( xiaomi 6x)

Android 调试桥 (adb) 是一种在PC端运行的命令行工具，可以通过命令与移动端设备进行通信，可连接真机和手机模拟器。

adb环境配置

1、下载Android SDK Platform Tools
https://developer.android.com/studio/releases/platform-tools

2、设置系统环境变量到该软件安装包，在命令运行界面输入adb看是否设置成功

3、手机通过数据线连接到电脑，需开启USB调试，如果是模拟器的话也需开启USB调试模式

4、在命令运行界面输入adb devices查看所有设备并进行连接调试，如果是模拟器需运行命令进行连接，如：adb connect 127.0.0.1:5555

screenrecord 实用程序的局限性：
1、音频不与视频文件一起录制。
2、无法在搭载 Wear OS 的设备上录制视频。
3、某些设备可能无法以它们的本机显示分辨率进行录制。如果在录制屏幕时出现问题，请尝试使用较低的屏幕分辨率。
4、不支持在录制时旋转屏幕。如果在录制期间屏幕发生了旋转，则部分屏幕内容在录制时将被切断。
