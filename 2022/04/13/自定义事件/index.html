自定义事件

发表于 2022-04-13 分类于 javascript

Event

new Event(type,eventInit);
type:创建事件的名称，DOMString类型
eventInit：可选配置(bubbles/cancelable/composed)

UIEvent

UIEvent 接口表示简单的用户界面事件，是从 Event 派生出来的，可以传detail字段（类型只能为数字,非数字时接收到的值为0）

CustomEvent

new CustomEvent(type,eventInit)
type:创建事件的名称，DOMString类型
eventInit：可选配置(bubbles/cancelable/detail)

Event、UIEvent、CustomEvent的区别

UIEvent、CustomEvent可以传递自定义参数detail，UIEvent detail类型只能为数字，CustomEvent可以是数字、字符串、数组、对象等

createEvent

注意：createEvent使用的方法很多方法类似initCustomEvent、initEvent已被废弃不建议使用，建议使用Event或createEvent代替
